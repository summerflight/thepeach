설치 및 세팅 방법

Windows 버전
https://github.com/ThePeach-Drone/qgroundcontrol/releases 페이지에서 릴리즈 버전 윈도우 설치 파일을 받은 후, 설치파일을 실행시켜 설치

Linux 버전
https://github.com/ThePeach-Drone/qgroundcontrol/releases 페이지에서 릴리즈 버전 Appimage를 다운받아 chmod +x ./Peach-GS.AppImage 명령어로 실행 권한을 부여하고, ./Peach-GS.Appimage로 실행



PEACH-GS 펌웨어 다운로드 및 설치하기


필요한 준비물

THE PEACH-FCC-K1(R1), Micro 5Pin 케이블, THE PEACH-GS를 설치한 컴퓨터


펌웨어 다운로드 및 설치 과정

1. Peach-GS 실행 후 ThePeach 아이콘 클릭

2. ThePeach 아이콘 클릭하여 메뉴 진입 후 Vehicle Setup 클릭


3. Vehicle Setup에서 Firmware 클릭


4. 컴퓨터에 연결된 마이크로 5핀 케이블과 더피치 FC를 연결

5. 아래와 같은 화면이 뜨는 것을 확인한 후, 보드 타입과 펌웨어를 선택한 후 OK 버튼 클릭


6. GS에서 자동으로 펌웨어 다운로드 및 보드 프로그래밍 진행 (설치 완료)
